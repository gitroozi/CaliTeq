## User Journey Maps

### Journey 1: New User Onboarding to First Workout

1. **Landing** → Create account
2. **Onboarding** → Complete assessment
3. **Plan Generation** → View 12-week plan
4. **Dashboard** → See today's workout
5. **First Workout** → Complete session
6. **Logging** → Track performance
7. **Completion** → View achievements

### Journey 2: Existing User Daily Workout

1. **Login** → Dashboard
2. **Today's Workout** → View exercises
3. **Start** → Begin session
4. **Log Sets** → Track each exercise
5. **Complete** → Submit workout
6. **Progress** → View updated metrics

### Journey 3: Progress Check-In

1. **Dashboard** → View progress
2. **Check-In** → Answer questions
3. **AI Analysis** → Receive recommendations
4. **Adjustments** → Review plan changes
5. **Continue** → Resume training
