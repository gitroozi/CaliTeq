## Frontend Component Architecture

### Page Components
- **Dashboard** - Main landing page
- **OnboardingFlow** - Multi-step assessment
- **WorkoutView** - Active workout session
- **ExerciseLibrary** - Browse exercises
- **ProgressDashboard** - Analytics and charts
- **Profile** - User settings

### Feature Components
- **WorkoutCard** - Today's workout display
- **ExerciseCard** - Exercise with video
- **SetLogger** - Log sets and reps
- **ProgressChart** - Data visualization
- **AchievementBadge** - Gamification elements

### Shared Components
- **Button** - Reusable button component
- **Input** - Form inputs
- **Modal** - Dialog overlays
- **Navbar** - Navigation header
- **Card** - Content containers
