## Onboarding Flow

The onboarding process collects comprehensive user data to generate optimal workout plans.

### Step 1: Personal Details
- Age, gender, height, weight
- Activity level
- Training experience

### Step 2: Goals
- Primary and secondary goals
- Goal prioritization
- Target metrics (weight, strength milestones)

### Step 3: Medical Screening
- Injury history
- Current limitations
- Exercise clearance status

### Step 4: Equipment & Availability
- Available equipment
- Training frequency (days/week)
- Session duration preference

### Step 5: Movement Assessment
- Optional video upload or self-assessment
- Current capability per movement pattern

### Output
- User profile with training classification
- Recommended starting point
- Contraindication flags
