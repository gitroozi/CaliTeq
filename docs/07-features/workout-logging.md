## Workout Logging

### Quick Log Interface
- Mark exercises as completed
- Log actual sets Ã— reps performed
- Rate difficulty (RPE 1-10)
- Note modifications or issues
- Track session duration
- Record energy level and overall feel

### Missed Workout Handling
- Options: Reschedule, skip, mark as rest day
- Impact on weekly volume tracking

### Data Collected Per Set
- Reps completed
- Target reps
- RPE (Rate of Perceived Exertion)
- Notes (optional)

### Aggregate Metrics
- Total reps (sum across sets)
- Max reps (best set)
- Average RPE
- Completion rate
